##安装过程
###需要先给termuxAPP所有文件访问权限
## termux 输入以下命令
#⚠️不管你之前有没有执行过以下命令，现在都必须执行
## 首次使用启动代码
```
(以下步骤打开VPN执行命令 否则无法下载WCTOOL文件)
```
```
rm -rf $HOME/WCTOOL
```
```
git clone --depth 1 https://github.com/wcbcbt/WCTOOL.git
```
```
chmod +x WC0-0/搭建代码
```
```
./WC0-0/搭建代码
```
## 第二次使用启动代码
```
cd WCTOOL
```
```
./WC1-10/WC
```
